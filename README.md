# CJCamPhish

Grab cam shots from target's phone front camera or PC webcam just by sending a link.

**Created by Coding Junction**

---

## What is CJCamPhish?

CJCamPhish is a tool that captures photos from the target's phone front camera or PC webcam. It hosts a fake website on an in-built PHP server and uses ngrok & CloudFlare Tunnel to generate a link. When the target opens the link and allows camera permission, their photo is captured.

GPS location tracking feature is also included.

- Festival Wishing template
